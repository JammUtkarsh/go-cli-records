
# CLI Data Entry App

A simple CLI app for data entry into a json file.

[Demo](https://localhost)

## Pre-requisite

Download [Go](https://go.dev/doc/install)

## Run Locally

Clone the project

```bash
  git clone https://github.com/JammUtkarsh/go-cli-records
```

Go to the project directory

```bash
  cd go-cli-records
```

Build executable

```bash
  go build
```

Run executable

```bash
  ./app
```
